🧮 Calculator Website

A simple and aesthetically pleasing calculator built using HTML5, CSS3, and JavaScript. This project combines a clean hand-drawn notebook-inspired design with essential arithmetic functionality, making it both visually engaging and easy to use. It supports basic mathematical operations with real-time expression display, keyboard shortcuts, and responsive button interactions for a smooth user experience.

✨ Features

Perform basic arithmetic operations: Addition (+), Subtraction (−), Multiplication (×), Division (÷), and Percentage (%)

Real-time expression and result display

AC (All Clear) and Backspace (⌫) functionality

Decimal number support

Keyboard input support for faster calculations

Responsive and user-friendly interface

Unique handwritten notebook-inspired UI design


🛠️ Technologies Used

HTML5 – Structure of the calculator

CSS3 – Styling, layout, animations, and responsive design

JavaScript (ES6) – Calculator logic and interactive functionality


🎯 Purpose

This project was created to strengthen front-end development skills by implementing JavaScript-based calculator logic while designing an attractive and intuitive user interface. It demonstrates proficiency in DOM manipulation, event handling, responsive web design, and interactive UI development.
